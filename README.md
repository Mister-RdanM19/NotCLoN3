# NotCLoN3 📝

**NotCLoN3** adalah aplikasi penyunting teks (*text editor*) ringan yang dirancang sebagai versi klon dari Notepad++. Aplikasi ini dibuat untuk memberikan pengalaman menulis kode atau catatan yang familiar namun dalam paket yang lebih personal.

---

## ✨ Fitur Utama
* **Ringan & Cepat:** Tidak memakan banyak memori, cocok untuk penggunaan harian.
* **Antarmuka Familiar:** Terinspirasi langsung dari Notepad++ untuk kemudahan navigasi.
* **Portable:** Cukup jalankan file `.exe` tanpa perlu instalasi rumit.

## 🚀 Cara Menjalankan
1.  Buka folder tempat Anda menyimpan repositori ini.
2.  Cari file `notclone.exe`.
3.  Klik dua kali untuk menjalankan aplikasi.

> [!NOTE]
> Karena ini adalah versi klon, beberapa fitur canggih dari Notepad++ asli mungkin masih dalam tahap pengembangan atau sengaja dibuat lebih sederhana.
